STEPS TO RUN THE PROJECT :-

1. Nodemon is required to be installed in your system.
2. Open the folder in VS Code.
3. Now to start the Server :-
In terminal type following command
            nodemon client.js
4. The server is now started.
5. Now open index.html file in your browser.
6. Register your name.
7. again open index.html in new window.
8. Your group chat is now ready ! It can add more than 2 users.
With features like notification sound,user add notification,user left notification.
